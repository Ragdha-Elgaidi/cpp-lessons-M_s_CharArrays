# cpp-lessons-M_s_CharArrays
## Escape characters are :
- \n = new line
- \t = tab
- \0 = null
- \” = “
### We can use them like that :
'''
#include<iostream>
using namespace std;

int main() {

	// Escape characters
	cout<<"hello\tworld\n";
	cout<<"\0";
	cout<<"Let's print a double quote \"  ";

	return 0;
}

'''
